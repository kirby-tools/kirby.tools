- Support [Anthropic Claude](https://www.anthropic.com/claude) models.

To use Claude Sonnet models, add the configuration for the `anthropic` provider:

```php [config.php]
return [
    'johannschopplich.copilot' => [
        'provider' => 'anthropic',
        'providers' => [
            'anthropic' => [
                'apiKey' => 'YOUR_API_KEY',
                'model' => 'claude-3-5-sonnet-20240620'
            ]
        ]
    ]
];
```

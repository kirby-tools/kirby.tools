**Initial Release**

Kirby Copilot v1.0 brings AI-powered content generation directly to the Kirby Panel. Generate text content from prompts using OpenAI's GPT models.

**ðŸš€ Features**

- **Panel section**: Add the Copilot section to any page blueprint to enable AI content generation. The section provides a prompt input field and generates content for your configured blueprint fields.
- **OpenAI integration**: Generate content using OpenAI GPT models. Configure your API key and preferred model in the [global configuration](/docs/copilot/configuration/global).
- **Customizable prompts**: Define [user prompts](/docs/copilot/configuration/local#userprompt) and [system prompts](/docs/copilot/configuration/local#systemprompt) to tailor AI responses for specific content types.
- **Field targeting**: Configure which fields the generated content should populate using the [`field`](/docs/copilot/configuration/local#field) or [`fields`](/docs/copilot/configuration/local#fields) section property.

For setup instructions, see the [Installation guide](/docs/copilot/getting-started/installation).

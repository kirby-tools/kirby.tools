## Translation Provider

At least one translation provider must be configured. Pick a backend below ŌĆō the [`strategy` option](#strategy) selects which one runs.

::card-group

:::card{title="DeepL" icon="i-simple-icons-deepl" to="/docs/content-translator/providers/deepl"}
Industry-leading machine translation with a free tier available.
:::

:::card{title="AI Translation" icon="i-ri-sparkling-line" to="/docs/content-translator/providers/ai-translation"}
Context-aware translation via the [Kirby Copilot](/copilot) plugin (OpenAI, Anthropic, Google, Mistral).
:::

:::card{title="Custom Strategy" icon="i-ri-puzzle-line" to="/docs/content-translator/providers/custom-translator"}
Plug in any translation API via a closure or a custom `Strategy` implementation.
:::

::

## Available Properties

Set global defaults that apply to both **view buttons** and **sections**. These can be overridden in individual blueprints. The following configuration sets DeepL as the translation provider and defines global defaults for field types, title translation, slug translation, confirmation dialogs, and KirbyTags translation:

```php [config.php]
return [
    'johannschopplich.content-translator' => [
        // Translation provider
        'DeepL' => [
            'apiKey' => env('DEEPL_API_KEY')
        ],

        // Global default properties for view buttons and sections
        'fieldTypes' => [
            'blocks',
            'text',
            'textarea'
        ],
        'title' => true,
        'slug' => true,
        'kirbyTags' => [
            'link' => ['text', 'title'],
            'image' => ['alt', 'caption'],
            'file' => ['text', 'title']
        ]
    ]
]
```

For detailed descriptions of each shared property, see the [View Button & Section Configuration](/docs/content-translator/configuration/local#available-properties) page. All local properties can be defined globally.

### `label` :u-badge{label="String" class="align-middle ml-2 rounded-full!" variant="subtle"}

Custom label for the Content Translator view button or section.

### `import` :u-badge{label="Boolean" class="align-middle ml-2 rounded-full!" variant="subtle"}

Enable or disable content importing functionality entirely.

### `importFrom` :u-badge{label="String" class="align-middle ml-2 rounded-full!" variant="subtle"}

Controls import direction. Set to `all` to allow importing from any language to any language.

### `batch` :u-badge{label="Boolean" class="align-middle ml-2 rounded-full!" variant="subtle"}

Enable or disable batch translation (translating to multiple languages at once).

### `title` :u-badge{label="Boolean" class="align-middle ml-2 rounded-full!" variant="subtle"}

Include the model title in import and translation operations.

### `slug` :u-badge{label="Boolean" class="align-middle ml-2 rounded-full!" variant="subtle"}

Include the model slug in import and translation operations. The slug will be generated from the translated title.

### `confirm` :u-badge{label="Boolean" class="align-middle ml-2 rounded-full!" variant="subtle"}

Show confirmation dialogs before import or translation operations.

### `fieldTypes` :u-badge{label="Array" class="align-middle ml-2 rounded-full!" variant="subtle"}

Specify which field types to include in import and translation operations.

### `includeFields` :u-badge{label="Array" class="align-middle ml-2 rounded-full!" variant="subtle"}

Specify specific fields to include in import and translation operations.

### `excludeFields` :u-badge{label="Array" class="align-middle ml-2 rounded-full!" variant="subtle"}

Specify fields to exclude from the import and translation process.

### `kirbyTags` :u-badge{label="Object" class="align-middle ml-2 rounded-full!" variant="subtle"}

Configure selective translation of KirbyTag attributes (e.g., link text, image alt text).

### `systemPrompt` :u-badge{label="String" class="align-middle ml-2 rounded-full!" variant="subtle"}

Override the AI translation system prompt. Takes precedence over the global `ai.systemPrompt` config option.

### `strategy` :u-badge{label="Mixed" class="align-middle ml-2 rounded-full!" variant="subtle"} :u-badge{label="v3.11+" class="align-middle rounded-full!" variant="subtle"}

Selects the translation backend explicitly. Accepts a string preset, a closure, or a [`Strategy`](/docs/content-translator/php-classes/strategies) instance. Without this option the plugin defaults to DeepL (and falls back to the deprecated `translateFn`).

| Value      | Resolves to                                                                                            |
| ---------- | ------------------------------------------------------------------------------------------------------ |
| `'deepl'`  | `DeepLStrategy` ŌĆō requires `DeepL.apiKey`                                                              |
| `'ai'`     | `CopilotAIStrategy` ŌĆō requires the [Kirby Copilot](/docs/copilot) plugin                               |
| `Closure`  | Wrapped in `CallableStrategy`. Signature: `fn (string $text, string $target, ?string $source): string` |
| `Strategy` | Used as-is ŌĆō instance of `JohannSchopplich\ContentTranslator\Translation\Strategy`                     |

::code-group

```php [String preset]
return [
    'johannschopplich.content-translator' => [
        'strategy' => 'deepl', // or 'ai'
    ]
];
```

```php [Closure]
return [
    'johannschopplich.content-translator' => [
        'strategy' => function (string $text, string $target, ?string $source): string {
            return myTranslateApi($text, $target, $source);
        },
    ]
];
```

```php [Custom Strategy]
use App\Translation\MyApiStrategy;

return [
    'johannschopplich.content-translator' => [
        'strategy' => new MyApiStrategy(),
    ]
];
```

::

### `batchConcurrency` :u-badge{label="Integer" class="align-middle ml-2 rounded-full!" variant="subtle"} :u-badge{label="v3.7.0+" class="align-middle rounded-full!" variant="subtle"}

In batch translation mode, multiple languages are translated in parallel to improve performance. By default, up to 4 translations are processed concurrently to avoid hitting API rate limits.

If you encounter DeepL API rate limit errors, you can reduce the concurrency by setting the `batchConcurrency` option in your global configuration. For example, to limit concurrency to 2:

```php [config.php]
return [
    'johannschopplich.content-translator' => [
        'batchConcurrency' => 2
    ]
]
```

### `ai.systemPrompt` :u-badge{label="String" class="align-middle ml-2 rounded-full!" variant="subtle"} :u-badge{label="v3.10.0+" class="align-middle rounded-full!" variant="subtle"}

When using AI translation, you can replace the built-in system prompt with a custom one. Set it globally or override it per blueprint.

```php [config.php]
return [
    'johannschopplich.content-translator' => [
        'ai' => [
            'systemPrompt' => 'You are a medical translator. Preserve clinical terminology and abbreviations.'
        ]
    ]
]
```

See the **AI Translation** docs for the full default prompt, blueprint override examples, and usage details.

## Get the Latest Version

### Via Composer :u-badge{label="Recommended" size="sm" class="align-middle ml-2 -translate-y-px rounded-full!" variant="subtle"}

::code-group

```bash [Install]
composer require johannschopplich/kirby-live-preview
```

```bash [Update]
composer update johannschopplich/kirby-live-preview
```

::

### Via ZIP Download

Download the :latest-version and extract it to your `site/plugins` folder. If updating, replace the existing plugin folder with the new one.

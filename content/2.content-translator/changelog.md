## Get the Latest Version

### Via Composer (Recommended)

::code-group

```bash [Install]
composer require johannschopplich/kirby-content-translator
```

```bash [Update]
composer update johannschopplich/kirby-content-translator
```

::

### Via ZIP Download

Download the :latest-version and extract it to your `site/plugins` folder. If updating, replace the existing plugin folder with the new one.

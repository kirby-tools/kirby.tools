**🐞 Bug Fixes**

- **AI translation reliability**: Provider artifacts no longer leak into stored content. Previously, some providers (most notably Mistral) could echo batch wrappers (`<item index="N">…</item>`), HTML entities, or refusal text. See the [documentation](/docs/content-translator/providers/ai-translation).

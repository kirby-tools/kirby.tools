**🚀 Features**

- **AI translations via Copilot's plugin API**: Content Translator now talks to Kirby Copilot through its versioned plugin API, keeping both plugins compatible as they evolve independently. Requires Kirby Copilot v3.9.0+ – older versions show an update notice when an AI translation is started. See the [AI translation documentation](/docs/content-translator/providers/ai-translation).

**🚀 Features**

- **Chat Completions mode for OpenAI-compatible gateways**: Set `providers.openai.api` to `chat` for endpoints that don't expose the Responses API. This unlocks Cloudflare AI Gateway's [Unified API](https://developers.cloudflare.com/ai-gateway/usage/chat-completion/) – route Gemini, Claude, and more through a single endpoint via the OpenAI SDK shape. See the [global configuration documentation](/docs/copilot/configuration/global#api).

**🔒 Security**

- **Proxy target allowlist**: The proxy now restricts upstream requests to known AI provider hosts plus any configured [`baseUrl`](/docs/copilot/configuration/global#baseurl). Prevents authenticated Panel users from redirecting traffic to attacker-controlled hosts.
- **Scoped API key forwarding**: API keys are now only injected into authentication headers. Prevents a Panel user from exfiltrating the key via other forwarded headers.

**🐞 Bug Fixes**

- **Proxy error handling**: Upstream provider errors and network failures now surface to the Panel with proper status codes and messages instead of a silent empty stream.

**🔄 Updated Default Models**

| Provider | Model      | Old          | New            |
| -------- | ---------- | ------------ | -------------- |
| OpenAI   | Completion | `gpt-5-nano` | `gpt-5.4-nano` |

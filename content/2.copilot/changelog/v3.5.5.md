Includes all fixes since v3.5.0.

**🐞 Bug Fixes**

- **Prompt editor unusable**: Fixed a Chrome-specific issue where the prompt editor couldn't be clicked or typed in when no initial prompt was set.
- **Cursor position**: The cursor now starts at the end of the text (matching native textarea behavior) when the prompt dialog opens with a pre-filled prompt.
- **Block field options**: Fixed a 400 error ("Invalid query result data: NULL") when block blueprints contain fields with query-based options.

- Support [Anthropic Claude](https://platform.claude.com/docs/en/about-claude/models/overview) models.

To use Claude Sonnet models, add the configuration for the `anthropic` provider:

```php [config.php]
return [
    'johannschopplich.copilot' => [
        'provider' => 'anthropic',
        'providers' => [
            'anthropic' => [
                'apiKey' => 'YOUR_API_KEY',
                'model' => 'claude-sonnet-4-20250514'
            ]
        ]
    ]
];
```

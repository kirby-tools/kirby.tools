::tip
Reasoning control, expanded: `reasoningEffort` now spans seven levels – from `none` to `xhigh` – and is translated to each provider's native reasoning controls automatically.
::

**🚀 Features**

- **Expanded `reasoningEffort` options**: New `provider-default`, `minimal`, and `xhigh` levels join the existing `none` / `low` / `medium` / `high`. Configure the effort once – it works across all reasoning-capable OpenAI, Google, Anthropic, and Mistral models. See the [Global Configuration documentation](/docs/copilot/configuration/global#reasoningeffort).
- **Plugin API**: Other plugins can drive Copilot's AI generation through a versioned Panel API – powering AI translations in [Kirby Content Translator](/content-translator) v3.12+.

**🔄 Updated Default Models**

| Provider  | Model      | Old                      | New                |
| --------- | ---------- | ------------------------ | ------------------ |
| OpenAI    | Generation | `gpt-5.4`                | `gpt-5.6-terra`    |
| Anthropic | Generation | `claude-sonnet-4-6`      | `claude-sonnet-5`  |
| Google    | Completion | `gemini-3-flash-preview` | `gemini-3.5-flash` |

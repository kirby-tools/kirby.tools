**🐞 Bug Fixes**

- **OpenAI-compat gateways – completion model fallback**: Inline suggestions no longer derive an unreachable default like `google-ai-studio/gpt-5.4-nano` when the configured `model` uses a cross-provider gateway prefix. An explicit `completionModel` is now required in this case and a clear error guides the configuration. See the [global configuration documentation](/docs/copilot/configuration/global#api).

::tip
Copilot's AI engine is now a first-class PHP API. Drive structured-output generation from CLI commands, model methods, or any custom workflow – the same provider stack that powers the Panel.
::

**🚀 Features**

- **PHP classes**: Generate text and structured JSON from any PHP context – no Panel required. See the [PHP classes documentation](/docs/copilot/php-classes).
- **`Client::generateObject()`**: Generate JSON objects conforming to a JSON Schema across all supported providers with a single API. See the [client reference](/docs/copilot/php-classes/client#generateobject).
- **`ProviderException`**: A dedicated exception class wraps upstream provider failures with `provider`, `httpCode`, and the original `Throwable`. See the [exceptions documentation](/docs/copilot/php-classes/exceptions).

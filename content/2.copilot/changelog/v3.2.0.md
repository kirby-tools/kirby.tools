- **Config-defined prompt templates**: Predefine prompt templates in the Kirby config that appear for all Panel users. Templates support multilingual labels and prompts via language arrays. Config templates are read-only and displayed alongside user-created templates. See the [Prompt Templates documentation](/docs/copilot/prompt-dialog/templates#config-defined-templates).

::code-group

```php [Basic]
return [
    'johannschopplich.copilot' => [
        'promptTemplates' => [
            [
                'label' => 'Apply House Style',
                'prompt' => 'Format the text according to our editorial style: artist names in bold, album titles in italics, use curly quotation marks.'
            ]
        ]
    ]
];
```

```php [Multilingual]
return [
    'johannschopplich.copilot' => [
        'promptTemplates' => [
            [
                'label' => [
                    'en' => 'Apply House Style',
                    'de' => 'Redaktionsstil anwenden'
                ],
```

Kirby Copilot v3 is a **free upgrade** for all users with a valid license key for v2. This major release brings significant security improvements and powerful new features.

- **Server-side API proxy**: All AI provider requests now route through a server-side proxy instead of direct client-side requests. Your API keys are no longer exposed in browser network requests, protecting them from Panel users.

- **Inline suggestions** in writer fields with dedicated [`completionModel`](/docs/copilot/configuration/global#default-models) configuration per provider. Press `Tab` to accept AI suggestions, `Escape` to dismiss. Ghost text appears after a brief pause. Manually trigger with `Cmd + ,` / `Ctrl + ,`. See the [Inline Suggestions documentation](/docs/copilot/usage/inline-suggestions).
- **Global `reasoningEffort`** configuration for all reasoning-capable models (GPT-5, Gemini 3, Claude 4). Controls the depth of reasoning during generation. See the [Global Configuration documentation](/docs/copilot/configuration/global#reasoningeffort).
- **Writer mark formatting preservation**: Replacing selected text now preserves all formatting (bold, italic, links, etc.). Generation supports all Kirby writer built-in marks.
- **Prompt preview**: Expand the **Preview** section in the prompt dialog to see [field placeholders](/docs/copilot/prompt-dialog/placeholders) replaced with their actual values before sending.
- **Prompt templates**: Save frequently used prompts as templates. Five built-in templates included (Fix Grammar, Make Concise, Simplify, Extend, Summarize). See the [Prompt Dialog documentation](/docs/copilot/prompt-dialog/templates).
- **Enhanced prompt history**: Access recent prompts via dropdown menu with arrow key navigation.

- **Temperature removal**: The `temperature` option has been removed. Modern reasoning models (GPT-5, Gemini 3, Claude 4) manage creativity internally. Use [`reasoningEffort`](/docs/copilot/configuration/global#reasoningeffort) instead.
